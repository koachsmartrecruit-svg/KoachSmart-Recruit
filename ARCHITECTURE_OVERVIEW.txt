ADMIN ROLE HIERARCHY
═══════════════════════════════════════════════════════════════════════════════

SUPER ADMIN (Platform Owner)
├── Create/Manage Regional Admins
├── Allocate Cities & Permissions
├── Monitor All Activities
├── Financial Oversight
├── System Configuration
└── View Analytics (All Cities)

REGIONAL ADMIN (City-Level)
├── Verify Coaches (Assigned Cities Only)
├── Approve/Reject Documents
├── Handle Local Issues
├── Monitor Local Metrics
├── Generate Local Reports
└── Activity Tracking (Audit Trail)

ONBOARDING vs VERIFICATION
═══════════════════════════════════════════════════════════════════════════════

ONBOARDING (Mandatory)          │  VERIFICATION (Optional)
├── Cannot Skip                 │  ├── For Higher Badges
├── 3 Steps                      │  ├── 4 Stages
├── 200 Coins Total              │  ├── 3500 Coins Total
├── Orange Badge                 │  ├── Purple, Blue, Green Badges
├── 15-30 Minutes                │  ├── Days/Weeks
├── Self-Reported                │  ├── Document Proof Required
└── Immediate Completion         │  └── Admin Review Required

Step 1: Personal Details (50 coins)
Step 2: Professional Details (100 coins)
Step 3: Location & Availability (50 coins)
TOTAL: 200 Coins + Orange Badge

├── Model: OnboardingProgress

Phase 3: Membership System (Week 3-4)
├── Models: MembershipPlan, UserSubscription, SubscriptionHistory
├── Routes: Plan selection, subscription management
├── Payment: Gateway integration
└── Dashboard: Subscription management, billing history

Phase 4: Multi-City Admin (Week 4-5)
├── Routes: Admin allocation, city assignment
├── Dashboards: Activity tracking, performance metrics
├── API: Key management
└── Middleware: City access control

Phase 5: Location Simplification (Week 5)
├── Forms: Updated onboarding forms
├── Service: Job matching algorithm
├── Database: Schema updates
└── Tests: Matching logic verification
